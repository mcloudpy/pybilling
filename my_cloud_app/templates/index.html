Mi Aplicación Cloud

PyBilling, monitorización y facturación sencilla para Cloud

Esto es una aplicación cloud de prueba

Esta aplicación pretende demostrar que es fácil integrar cualquier aplicación con PyBilling, de tal forma que sea posible medir el uso que se hace de dicha aplicación, ya sea mediante 3scale o mediante un sistema propio.

Para usar esta aplicación haz click en el botón. Se hará uso de mCloud PyBilling para registrar el uso, de tal forma que los administradores de la aplicación puedan gestionar la información de facturación.

Para gestionar dicha información existen dos opciones provistas por mCloud PyBilling

  • 3scale
  • Sistema propio
